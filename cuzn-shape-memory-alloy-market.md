<p><strong>The CuZn Shape Memory Alloy Market Analysis by types is segmented into:</strong></p>
<p><ul><li>One-Way Memory Effect</li><li>Two-Way Memory Effect</li></ul></p>
<p><p>The CuZn Shape Memory Alloy market is characterized by two main types: One-Way Memory Effect and Two-Way Memory Effect. The One-Way Memory Effect allows the material to return to a predetermined shape upon heating, useful in applications like stents and actuators. In contrast, the Two-Way Memory Effect enables the material to switch between two shapes with temperature changes, making it ideal for applications where consistent shape change is required, such as smart textiles and sensors.</p></p>
